# 智能称重助手

## ✨ 介绍

本小程序基于微信开发者工具开发，使用**智能物联设备（蓝牙称重器） + 微信小程序**的方式，替代传统农业称重场景中人工收集数据、统计数据的低效问题。

## 🕹️ 硬件设备

目前小程序仅支持和以下设备进行蓝牙连接，并通过蓝牙串口实时获取称重数据（其它设备未经测试）

- [德力 DELI-001](https://e.tb.cn/h.SEYzRW5NpcXFPAQ?tk=GmnXfl9A7NT)

## 📄 页面

### 1. 称重记录(scale)

首页默认显示称重模式：工人扫码识别和称重器串口连接两个模块

### 2. 工人记录(workers)

工人信息的CRUD界面

### 3. 个人信息(home)

注册、显示个人信息
